目前moyuAdmin提供的功能有`系统功能`和`系统扩展功能`。
* `系统功能`是系统运行和管理的基础功能，是系统必需的最小功能集
* `系统扩展功能`是对系统基本能力的扩展，类似于插件形式，如不需要删除即可。

## 系统功能
系统运行必不可少的功能，有了这些基础功能就可以运维和管理。
主要包括3类7个菜单。

* 组织架构：用于对组织机构和人员的管理
  * `组织管理`
  * `用户管理`
* 权限管理：用于对系统进行权限控制
  * `岗位权限`
  * `角色管理`
* 资源管理：用于对目录、菜单、按钮、接口等资源的管理
  * `模块管理`
  * `菜单管理`
  * `接口管理`

## 系统扩展功能
系统功能的补充和完善，可按需使用，不需要可删除。
当前主要包括：

* `站内信`
* `代码生成`
* `系统日志`
* `会话管理`
