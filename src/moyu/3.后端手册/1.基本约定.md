## 数据格式

通过`BaseResponse、PageResult`定义了数据交互的基本格式

::: tabs

@tab 基本响应
```json
{
    "code": 0,
    "message": "成功",
    "data": null
}
```

@tab 分页请求
```json
{
    "pageNum": 1,
    "pageSize": 10
}
```

@tab 分页响应
```json
{
    "code": 0,
    "message": "成功",
    "data": {
        "total": 20,
        "records": []
    },
}
```
:::


## 异常约定

通过`BaseException、ExceptionEnum`定义了基本异常和错误码。

> 0: 成功
> -1: 参数错误。
> 1: 业务异常。用于可预期的但无法继续进行业务操作的异常。(前端不可直接展示)
> 2: 系统异常。用于无预期的、未知的异常。(前端不可直接展示)
> 3: 自定义异常。用于自定义提示信息的异常，前端可直接展示对应的提示信息。

使用时可以抛出`BaseException`异常。
```java
throw new BaseException(ExceptionEnum.INVALID_PARAMETER, "xxx不能为空");
```
