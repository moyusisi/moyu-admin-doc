## 数据格式

通过`BaseResponse、PageResult`定义了数据交互的基本格式

::: code-tabs

@tab 基本响应
```js
{
    "code": 0,
    "message": "成功",
    "data": null
}
```

@tab 分页请求
```js
{
    "pageNum": 1,
    "pageSize": 10
}
```

@tab 分页响应
```js
{
    "code": 0,
    "message": "成功",
    "data": {
        "total": 20,
        "records": []
    },
}
```
:::
