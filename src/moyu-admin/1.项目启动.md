## 环境准备
本项目所需环境既有后端[moyu-boot]()环境依赖，又有前端[moyu-antdv-admin]()的环境依赖。

## 启动项目
改造子应用的入口文件，暴露 `qiankun` 的生命周期钩子，实现子应用的挂载/卸载逻辑

### 拉取代码

```shell
git clone https://github.com/moyusisi/moyu-admin.git
cd moyu-admin
```

### 初始化准备
打包前要确保所需环境已就绪：
- 确保前端环境的Node.js、npm等工具准备就绪。
- 确保后端环境的JDK、Maven等工具准备就绪。

运行前要确保所需依赖已就绪：
* 确保后端依赖的中间件准备就绪，如MySQL、Redis等，确保数据库用户名/密码正确，数据库脚本执行完毕。

### 打包

```shell
# 确保进入项目目录
cd moyu-admin
# 打包
./mvnw clean package
```

### 运行

打包结束之后,`target`目录下会生成一个可执行jar包`moyu-admin.jar`，可以使用以下命令运行：

```shell
# 运行jar包
java -jar moyu-admin.jar
# 指定运行参数
java -jar moyu-admin.jar --spring.profiles.active=local
```
